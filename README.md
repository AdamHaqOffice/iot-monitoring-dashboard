# IoT Monitoring Dashboard

A premium React dashboard concept for an IoT monitoring product. This project was created as a portfolio piece for Adam Haq to showcase Python/Django-style product thinking, telemetry visualization, alerting, and device monitoring workflows.

## Stack

- React
- Vite
- Plain CSS

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Suggested portfolio positioning

Use this as a featured project that represents:

- Python / Django application design
- IoT monitoring experience
- dashboard UX
- production-minded system visualization
